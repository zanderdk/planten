Ciphers

With curl's options CURLOPT_SSL_CIPHER_LIST and --ciphers users can control which ciphers to consider when negotiating TLS connections.

TLS 1.3 ciphers are supported since curl 7.61 with options CURLOPT_TLS13_CIPHERS and --tls13-ciphers The names of the known ciphers differ depending on which TLS backend that libcurl was built to use. This is an attempt to list known cipher names.
