6 Enables Negotiate (SPNEGO) authentication.
7 
8 This option requires a library built with GSS-API or SSPI support. Use
9 --version to see if your curl supports GSS-API/SSPI or SPNEGO.
10 
11 When using this option, you must also provide a fake --user option to activate
12 the authentication code properly. Sending a '-u :' is enough as the user name
13 and password from the --user option aren't actually used.
14 
15 If this option is used several times, only the first one is used.
