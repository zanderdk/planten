1 Long: noproxy
2 Arg: <no-proxy-list>
3 Help: List of hosts which do not use proxy
4 Added: 7.19.4
5 ---
6 Comma-separated list of hosts which do not use a proxy, if one is specified.
7 The only wildcard is a single * character, which matches all hosts, and
8 effectively disables the proxy. Each name in this list is matched as either
9 a domain which contains the hostname, or the hostname itself. For example,
10 local.com would match local.com, local.com:80, and www.local.com, but not
11 www.notlocal.com.
12 
13 Since 7.53.0, This option overrides the environment variables that disable the
14 proxy. If there's an environment variable disabling a proxy, you can set
15 noproxy list to \&"" to override it.
