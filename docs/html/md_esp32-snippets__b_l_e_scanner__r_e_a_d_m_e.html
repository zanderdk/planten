BLE GAP DUMP

Code updated to support SDK v2.1

Supports dumping of Bluetooth GAP frames. For more information about Bluetooth Generic Access Profile

It will dump the header and then start dumping the payload. Also added is decoding of some payloads frames

In addition to dumping raw frames, the code supports decoding of some frames. Frames that support decoding:

  • 0x16, Subtype 0x1809 Temperature
  • 0x16, Subtype 0x180F Battery
  • 0x09, Complete local name
